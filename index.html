Página 1: Inicio — ¿Qué ocurre al ingresar una URL?

El proceso inicia cuando una persona escribe una dirección web en la barra del navegador. Esa dirección se llama URL y sirve para indicar qué recurso se quiere abrir.

Usuario → URL → Navegador

Partes de una URL URL HTTP Web

Elemento Descripción Ejemplo
Protocolo Indica cómo se comunica el navegador. https://
Dominio Nombre del sitio web. www.ejemplo.com
Ruta Ubicación del recurso. /index.html

Proceso

  • El usuario escribe una URL.
  • El navegador identifica el sitio solicitado.
  • Luego se prepara una petición HTTP.

💡 Recordatorio

La URL no muestra la página por sí sola. Primero el navegador envía una petición HTTP al servidor y luego recibe la información para mostrar la página web.
